# Building CSP for Windows


> ⚠️ **Warning:** This build system is still in development. We do not advise you use this for development.


This page details how to build CSP Visual Studio on Windows using the new conan + cmake build system.

## Prerequisites

This assumes you have CMake > 3.31, conan2 with both added to PATH.

https://cmake.org/download/

https://docs.conan.io/2/installation.html

## Build Instructions
Follow the steps below to build the **Connected Spaces Platform** project:

1. Clone the `connected-spaces-platform` remote repository by running the Git command `git clone https://github.com/magnopus-opensource/connected-spaces-platform.git`.

2. Open a Command Line Prompt from your `connected-spaces-platform` local repository.

3. Install our dependencies using Conan:

    `conan install . -s build_type=Debug --build=missing --profile:host=profiles/host/osx_arm`

    If you also want to mmake release builds in your ide, you will also have to download the release dependencies by changing build_type to Release.

    If you want to build a shared library, append: `-o connected-spaces-platform/*:shared=True`

4. Configure CMake, the most simple way of doing this is uing the preset files generated by conan:
    
    `cmake --preset conan-default`
    
    > ⚠️ **Note:** Some generators use different names for presets. Check the build folder if conan-default doesn't exst.


5. Open build/Debug/CSP.xcodeproj
